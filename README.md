# Portfolio Repository

This repository houses multiple versions of my portfolio projects in plain subfolders:

- `current/`: My current portfolio.
- `v1/`: First Porfolio (Vanilla HTML, CSS and JavaScript).
- `v2/`: Second Portfolio (Next.js).

## Development

To work on a specific version, navigate into its folder and run the dev script:

### Version 2
```bash
cd v2
npm install
npm run dev
```

### Active Version (`current`)
```bash
cd current
